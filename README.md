## Installation

Follow these steps to set up the To-DoList application locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/amavicpos/todo-react.git
   ```

2. Run:
   ```bash
   npm run start
   ```
or
   ```bash
   react-scripts start
   ```
